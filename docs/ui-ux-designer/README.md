## 🎯 设计原则

### 核心风格
- **风格类型**: Glassmorphism + Dark Mode (OLED)
- **设计理念**: Dark tech colors + trust + vibrant accents

### 色彩亮点
- 🎯 专业Fintech配色（信任感+科技感）
- 🌟 OLED优化（纯黑背景）
- 🔆 霓虹色彩系（增强视觉吸引力）

## 🎨 简化版色彩系统

```css
:root {
  /* Fintech主色调 */
  --color-primary: #F59E0B;         /* 主色 */
  --color-secondary: #FBBF24;       /* 辅助色 */
  --color-tertiary: #8B5CF6;       /* 强调色 */

  /* 分类标签颜色 */
  --category-finance: #FFD700;   /* 财经类 - 金黄 */
  --category-domestic: #4A90E5;    /* 国内政治 - 蓝色 */
  --category-international: #8B5CF6; /* 国际 - 紫色 */
  --category-general: #999999;    /* 普通 - 灰色 */

  /* 重要性标记颜色 */
  --importance-1: #CCCCCC;      /* 1星 - 浅灰 */
  --importance-2: #AAAAAA;      /* 2星 - 灰 */
  --importance-3: #F59E0B;      /* 3星 - 主色 */
  --importance-4: #FBBF24;      /* 4星 - 辅助色 */
  --importance-5: #FFD700;      /* 5星 - 金黄 */

  /* 功能色 */
  --color-success: #39FF14;
  --color-warning: #FFD700;
  --color-danger: #F59E0B;
}
```
