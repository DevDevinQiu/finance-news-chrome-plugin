## 📋 简化版需求规格

### 1.1 项目概述

**项目名称**: AI金融新闻推送 Chrome 插件（简化版）

**目标用户**:
- 散户投资者
- 金融从业者
- 对财经新闻感兴趣的普通用户

**核心价值主张**:
- **效率提升**：自动聚合多个财经新闻源，节省浏览时间
- **实时性**：定时更新（默认5分钟，用户可配置1-60分钟）
- **便捷性**：右侧面板随时查看，不打扰正常浏览

**变更说明**：
- ❌ 移除：AI股市预测功能（开盘预测、尾盘建议、置信度显示）
- ❌ 移除：AI分析理由展示
- ✅ 保留：核心数据聚合和推送功能

### 1.2 功能需求

#### 核心功能（MVP - 必须实现）

**新闻聚合与展示**
- [ ] 从3个数据源抓取财经新闻
- [ ] 实时更新新闻列表（默认5分钟，用户可配置1-60分钟）
- [ ] 按时间倒序展示
- [ ] 显示标题、来源、发布时间、分类标签
- [ ] 自动去重（24小时内重复新闻不显示）

**新闻分类**
- [ ] 财经类（黄色标签）
- [ ] 国内政治类（蓝色标签）
- [ ] 国际新闻类（紫色标签）
- [ ] 普通新闻（灰色标签）

**搜索与筛选**
- [ ] 标题关键词搜索（实时结果更新）
- [ ] 按分类筛选（单选）
- [ ] 按时间范围筛选（今天/本周/本月/全部）
- [ ] 快速筛选按钮

**重要性标记**
- [ ] 基于关键词匹配自动计算重要性
- [ ] 基于来源权威性调整权重
- [ ] 可视化标记（星级或颜色深浅）

**用户界面**
- [ ] 右侧可折叠面板（380px宽度）
- [ ] 点击展开/收起按钮
- [ ] 平滑动画效果（150-300ms）
- [ ] 固定在浏览器右侧

**Popup配置页面**
- [ ] 基础统计卡片（新闻总数、最后更新时间）
- [ ] 更新间隔滑块（1-60分钟）
- [ ] 数据源开关（启用/禁用）
- [ ] 保存按钮

#### 移除的功能

- [ ] ❌ AI股市预测（9:00开盘预测）
- [ ] ❌ AI尾盘操作建议（14:00建议）
- [ ] ❌ AI置信度显示
- [ ] ❌ AI分析理由展示
- [ ] ❌ 数据可视化图表
- [ ] ❌ 准确率统计

### 1.3 性能需求

#### 启动性能
- [ ] 插件启动时间 < 3秒
- [ ] 面板首次展开时间 < 500ms

#### 运行性能
- [ ] 面板展开/收起动画流畅（150-300ms）
- [ ] 搜索响应时间 < 300ms
- [ ] 筛选操作响应 < 200ms
- [ ] 内存占用 < 50MB

#### 数据性能
- [ ] 新闻列表滚动流畅（100+条新闻无卡顿）
- [ ] 大量新闻加载优化（虚拟列表或分页）
- [ ] 搜索结果即时显示
